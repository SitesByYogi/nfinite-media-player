== Usage ==
1) Add Tracks (CPT: Tracks) with Audio Preview URL + Product URL.
2) Create Playlists (CPT: Playlists) and add tracks, drag to reorder.
3) Shortcodes:
   - [nfinite_media_player ids="12,34" theme="dark" buy_label="Purchase" show_meta="yes"]
   - [nmp_playlist id="123" theme="dark" buy_label="Purchase" show_meta="yes"]
4) Elementor widget: choose Source (Tracks or Playlist), then style in the Style tab (colors, radius, spacing, buy button).
